Couloir 4

Les compétitions de natation des maîtres

À propos

Couloir 4 recense les compétitions de natation maîtres en France sur une carte interactive. Les données proviennent de liveFFN.

Si une compétition n'apparaît pas sur Couloir 4, ce n'est pas un bug : c'est qu'elle n'est pas encore sur liveFFN.

Si vous souhaitez faire apparaître une compétition sur Couloir 4, demandez à votre club de l'ajouter sur liveFFN et de mettre le mot « maîtres » dans le titre. Elle apparaîtra 24 h après sur Couloir 4.

Si une compétition maîtres existe sur liveFFN mais pas sur Couloir 4, soit vous n'avez pas attendu 24 h, soit le mot « maîtres » ou « master » ne figure pas dans le titre sur liveFFN. Dans ce cas, envoyez-nous une demande via ce formulaire.

Données & vie privée

Couloir 4 ne dépose aucun cookie et n'utilise aucun tracker publicitaire.

Statistiques de fréquentation. Le site utilise GoatCounter (couloir4.goatcounter.com), un outil d'analyse sans cookie. Les données collectées sont anonymisées : page vue, référent, pays approximatif, type de navigateur. Aucune adresse IP ni identifiant personnel n'est conservé. Base légale : intérêt légitime (la CNIL exempte de consentement les outils de mesure d'audience non-traçants).

Stockage local. Votre dernière position géographique, utilisée pour calculer les distances aux compétitions, est conservée uniquement dans votre navigateur via localStorage (clé couloir4.userPos.v1). Elle n'est jamais transmise à un serveur. Vous pouvez l'effacer à tout moment via les paramètres de votre navigateur.

Vos droits. Aucune donnée personnelle n'étant collectée par l'éditeur, il n'y a pas de demande à formuler. Pour toute question, vous pouvez contacter l'éditeur via Instagram ou LinkedIn.

Sources & crédits

Les données de compétitions proviennent de liveFFN, site de la Fédération Française de Natation. Couloir 4 est un projet indépendant et non-officiel, sans aucun lien avec la FFN.

Note aux administrateurs de liveFFN

Couloir 4 consulte liveFFN pour récupérer le calendrier des compétitions maîtres et les tailles de bassin. Le trafic est volontairement ultralight : 2 passages automatisés par jour (vers 11 h et 17 h, moins de 10 pages visitées à chaque passage).
